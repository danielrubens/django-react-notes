# Django React Notes

## About
A [Django](https://www.djangoproject.com/) project using [React](https://facebook.github.io/react/) on the client side.

## Start server
```
cd server
python -m venv venv
pip install -r requirements.txt
source venv/bin/activate
cd notes
python manage.py runserver
```

## Start client
```
cd client
npm install
npm start
```
